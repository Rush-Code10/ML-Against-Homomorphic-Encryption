1. Encrypted Workloads

The system executes mean, variance, dot product, linear regression inference, and logistic-regression approximation over encrypted inputs.

2. Metadata Capture

No plaintext is exposed. The collector records runtime, ciphertext size proxy, operation count, multiplicative depth, and a noise-budget-style signal.

3. Adversarial Inference

The ML models treat metadata as the attack surface and learn to classify which hidden encrypted computation produced each trace.
